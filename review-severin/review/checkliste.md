## Konventionen

### Use Intention-Revealing Names

Namen (Klassen, Funktionen, Variablen) sollen den Verwendungszweck abbilden

| Filename        | Kommentar                                  | Status |
|-----------------|--------------------------------------------|--------|
| Aufgabe.java    | -                                          | Ok     |

| Bild.java       | -                                          | Ok     |

| SharedList.java | -                                          | Ok     |
